SISTEMA DE GESTION DE PRACTICAS PROFESIONALES
=============================================
Proyecto Java (Ant) + JavaFX 8, arquitectura MVC.
Autor: Jorge Iran Perez Segovia

REQUISITOS
----------
- JDK 1.8 (incluye JavaFX 8). En este equipo: /usr/lib/jvm/jdk1.8.0_202
- Apache NetBeans (con soporte de proyectos Ant "Java with Ant").
- MySQL en localhost:3306 con la base de datos y el usuario configurados
  (ver seccion BASE DE DATOS).

LIBRERIAS
---------
- lib/mysql-connector-j-8.2.0.jar  -> conector JDBC de MySQL, ya incluido en el
  classpath del proyecto (nbproject/project.properties -> javac.classpath).
  NetBeans lo muestra en el nodo "Libraries" del proyecto.

ABRIR EN NETBEANS
-----------------
1. File > Open Project... y seleccione esta carpeta
   (SistemaGestionPracticasProfesionales). NetBeans la reconoce como proyecto
   Ant por la carpeta nbproject/.
2. El proyecto usa la plataforma de Java llamada "JDK_1.8". Si NetBeans muestra
   un problema de plataforma, regístrela en:
   Tools > Java Platforms > Add Platform...  y apunte a la carpeta del JDK 8
   (por ejemplo /usr/lib/jvm/jdk1.8.0_202). Nómbrela "JDK_1.8".
   (La ruta también está fijada en nbproject/project.properties como
    platforms.JDK_1.8.home, por lo que la compilación por consola ya funciona.)
3. Clic derecho en el proyecto > Run.

COMPILAR / EJECUTAR POR CONSOLA (Ant)
-------------------------------------
   ant clean jar     -> compila y genera dist/SistemaGestionPracticasProfesionales.jar
   ant run           -> ejecuta la aplicacion

BASE DE DATOS
-------------
El sistema usa la base de datos MySQL "practicas_sgpp" con el usuario
"practicas_app". Para prepararla desde cero:

   1) Crear base de datos y usuario (como administrador root):
        mysql -u root -p < crear-usuario-bd.sql
   2) Crear las tablas y cargar datos de ejemplo:
        mysql -u practicas_app -p practicas_sgpp < esquema-bd.sql

Parametros de conexion (editables, NO hardcodeados): src/configuracion-bd.properties
   base de datos : practicas_sgpp     usuario : practicas_app

ESTRUCTURA (MVC)
----------------
   src/sgpp/SistemaGestionPracticasProfesionales.java  -> clase principal
   src/sgpp/controlador/   -> controladores (sufijo Controller en ingles)
   src/sgpp/modelo/pojo/    -> POJOs (entidades del dominio)
   src/sgpp/modelo/dao/     -> DAO (acceso a datos: UsuarioDAO, ProyectoDAO, etc.)
   src/sgpp/modelo/ConexionBD.java     -> conexion a MySQL (formato estandar)
   src/configuracion-bd.properties     -> parametros de conexion (no hardcodeados)
   src/sgpp/vista/          -> vistas FXML (sin hojas CSS; estilos en linea)
   src/sgpp/utilidad/       -> utilidades de navegacion y alertas
   crear-usuario-bd.sql / esquema-bd.sql -> scripts de base de datos

NOTA: El nombrado evita abreviaciones y usa espanol; solo las clases
controladoras conservan el sufijo "Controller" en ingles.
